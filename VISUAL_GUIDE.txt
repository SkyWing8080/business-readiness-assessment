## What Each File Does

### app/api/submit-assessment/route.js
- Receives form submissions
- Validates data
- Saves to Postgres
- Sends email via Resend
- Returns success/error

### app/layout.js
- Minimal Next.js layout requirement
- Sets page metadata

### app/page.js
- Root page that redirects to `/assessment.html`
- Provides fallback if someone visits `/`

### lib/db.js
- Database connection functions
- Saves leads to Postgres
- Queries lead data
- Updates email tracking

### lib/email-templates.js
- Welcome email template (Email 1)
- Analysis email template (Email 2)
- Roadmap email template (Email 3)
- Professional HTML email design

### package.json
- Lists dependencies (Next.js, Postgres, Resend)
- Defines build scripts
- Specifies Node.js version

### next.config.js
- Next.js configuration
- Sets up URL rewrites
- Optimizes for production
